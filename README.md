# Online Course Platform

Современная образовательная платформа, разработанная на Django, предназначенная для онлайн-обучения учеников и взаимодействия с преподавателями.

---

## Возможности

### Для учеников

- Регистрация в несколько этапов
- Авторизация
- Выбор предметов
- Выбор преподавателей
- Просмотр доступных уроков
- Просмотр встроенных YouTube-видео
- Загрузка домашних заданий
- Просмотр своего профиля

### Для преподавателей

- Просмотр закрепленных учеников
- Проверка домашних заданий
- Оценивание работ
- Оставление комментариев

### Для администратора

- Управление пользователями
- Добавление предметов
- Создание уроков
- Добавление видео
- Управление домашними заданиями
- Загрузка PDF-материалов

---

# Используемые технологии

- Python
- Django
- SQLite (или MySQL)
- HTML5
- CSS3
- JavaScript
- Bootstrap
- YouTube Embed API
